# Multiple Linear Regression Assignment

This repository contains a Multiple Linear Regression analysis performed
on the Toyota Corolla dataset as part of a data Science practical assignment.

The objective of this assignment is to predict the price of a car using
multiple independent variables.

## Steps Covered
- Loading and understanding the dataset
- Exploratory Data Analysis (EDA)
- Handling categorical variables using encoding
- Feature scaling
- Train-test split
- Building Linear Regression model
- Applying Ridge and Lasso regression
- Model evaluation using MSE and R² score

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab


## Conclusion
Multiple Linear Regression models were successfully implemented.
Regularization techniques such as Ridge and Lasso helped improve
model stability and performance.
